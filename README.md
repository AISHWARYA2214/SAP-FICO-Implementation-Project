# SAP-FICO-Implementation-Project
"End-to-End SAP S/4HANA Finance Configuration for a dummy Company."
## Project Overview
This project covers end-to-end SAP FICO configuration including:

- Enterprise Structure
- General Ledger
- Accounts Payable
- Accounts Receivable
- Asset Accounting
- Cost Center Accounting

## Tools Used
- SAP ECC / S4 HANA
- MS Excel
- MS Word

## Key Transactions Used
- FB60
- FB70
- F-28
- F-53
- F-90
- AFAB

## Reports Generated
- Trial Balance
- P&L Statement
- Balance Sheet
